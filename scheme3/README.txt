HOIL7 Scheme 3 - Film Strip Split

Files:
- index.html
- scheme3.css
- scheme3.js

How to use:
1. Open your project folder.
2. Go to: scheme3/
3. Replace the existing index.html, scheme3.css and scheme3.js with these files.
4. Start local server from the project root:
   npx serve . -l 5500
5. Open:
   http://localhost:5500/scheme3/

What changed:
- Scheme 3 is now separated cleanly into HTML, CSS and JS.
- The infinity profile film-strip preview is now the actual Scheme 3 page.
- Hover/tap on a profile card shows glow, name, age and quote.
- Click opens Scheme 4 in a new tab with URL parameters:
  ../scheme4/?from=scheme3&userId=...&person=...&age=...
